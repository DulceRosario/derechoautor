Procedimiento para el registro de Software en el CCA
=========================================================

1.- **LLenar cesión de derechos a la UNAM.** Cada uno de los Autores deberán llenar un formato para ceder los derechos del programa a la UNAM. 

        Formato para autores que pertenecen a la UNAM. (:download:`Descargar <Forms/Cesion_SCIEM.doc>`)

        Formato para autores ajenos a la UNAM. (:download:`Descargar <Forms/Cesión de_derechos_persona_ajena_a_la_UNAM.doc>`)

2.-**Realizar la desccripción del Software**.  

3.- **LLenar formato con datos de los autores.** Cada uno de los autores debe llenar un formato con sus datos personales. (:download:`Descargar <Forms/02_Datos requeridos_por_INDAUTOR.doc>`)

4.- **Entregar  Ivonne** 
            
            * CD con el programa a registrar (3 copias)
            
            * Impresión del Código fuente (3 copias)

            * Cesión de derechos (1 por cada autor)
            
            * Formato con los datos generales de los autores (1 por cada autor)
            
            * Documento con la descripción del programa
